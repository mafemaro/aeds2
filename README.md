## 💻 Como Programar em Java

### 🔹 Usando VS Code

#### Pré-requisitos
- Java JDK 17 ou superior
- Visual Studio Code
- Extensão **Extension Pack for Java**

#### Verificando instalação do Java
```bash
java -version
```

#### Compilar e executar um programa Java

```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```

### 🔹 Usando IntelliJ

1. Instale o IntelliJ IDEA (Community Edition)
2. Crie um novo projeto Java
3. Selecione o JDK instalado
4. Crie uma classe Main
5. Execute com Run
```bash
✔ Recomendado para projetos maiores e organização por pacotes
```
